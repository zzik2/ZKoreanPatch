## 사용 방법:
1. ``%appdata%/Hytale/UserData/Mods`` 폴더에 플러그인 파일을 넣기
2. 기존 월드나 새로운 월드에서 플러그인 활성화
3. 플러그인이 활성화된 월드에 접속 후 게임 재시작(1회성) 🚀
